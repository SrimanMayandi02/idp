# idp

Demo of a globally-installable Go CLI. When run, it prints an "IDP" ASCII banner.

## Install

### Homebrew (macOS / Linux)

```bash
brew install SrimanMayandi02/tap/idp
```

### Go

```bash
go install github.com/SrimanMayandi02/idp@latest
```

### curl (Linux / macOS)

```bash
curl -fsSL https://raw.githubusercontent.com/SrimanMayandi02/idp/main/install.sh | sh
```

### Docker

```bash
docker run --rm ghcr.io/SrimanMayandi02/idp
```

### Direct download

Grab the binary for your OS from the [Releases page](https://github.com/SrimanMayandi02/idp/releases).

## Run

```bash
idp
```
